# 重庆旅游攻略 - 2025国庆完整指南

一个精美的重庆旅游攻略网站，包含详细的景点介绍、行程规划、交通指南和美食推荐。

## 特色功能

- 🏔️ 精美的视觉设计和动画效果
- 📱 响应式布局，支持移动端
- 🗺️ 详细的3天行程规划
- 🚇 完整的交通出行指南
- 🍲 地道的重庆美食推荐
- 🏨 住宿和购物建议

## 技术栈

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript
- Font Awesome 图标

## 本地运行

```bash
# 启动本地服务器
python3 -m http.server 8000

# 或使用 Node.js
npx serve .
```

然后访问 `http://localhost:8000`

## 部署

本项目已配置好 Vercel 部署，只需连接 GitHub 仓库即可自动部署。

```

```
